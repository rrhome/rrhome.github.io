    2019-09-23
    • 经验分享

    必装的扩展及其用途

    Markdown All in One

    提升了 VS Code 内置编辑器功能，快捷键、自动完成、编辑等等，最主要是加入了对数学公式的显示支持。

    Markdown Extended

    增强了粘贴、格式化表格等编辑功能，最主要的是它提供了所见即所得 (what you see is what you get) 的导出功能，支持导出为 HTML/PDF/PNG/JPEG 格式。

    Markdown Paste

    强化的粘贴功能，使用Ctrl+Alt+V。

    Marddown preview Mermaid Support

    使 VS Code 的预览支持 Mermaid 语法的图表和流程图。

    Graphviz Markdown Preview

    支持使用 Graphviz 语法的图像预览功能。

    markdownlint

    语法检测和提示。

    Markdown Preview Github Styling

    让预览看起来有 Github 的风格，可以根据喜好安装。
